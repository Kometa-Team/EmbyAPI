Overview
----------------------------------------------------------
Unofficial Python bindings for the Emby API. The goal is to make interaction with the API as easy as possible while emulating the Web Client as much as possible

Installation & Documentation
----------------------------------------------------------

.. code-block:: python

    pip install embyapi
